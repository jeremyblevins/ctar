# Changelog

All notable changes to the Cyber Threat Assessment Report (CTAR) Framework will be documented in this file.

The project follows Semantic Versioning.

## [1.0.0] - 2026-07-01

### Added
- Initial CTAR Framework release.
- Methodology Specification (CTAR-MS)
- Governance Charter (CTAR-GC)
- Style Guide (CTAR-SG)
- Code of Ethics & Analytical Conduct (CTAR-CE)
- Artificial Intelligence Participation Policy (CTAR-AIPP)
- Data Authenticity Policy (CTAR-DAP)
- Source Reliability Matrix (CTAR-SRM)
- Publication Safety Review (CTAR-PSR)
- Report Template (CTAR-RT)
- Initial historical CTAR example: ILOVEYOU.
