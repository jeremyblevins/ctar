# CTAR Roadmap

## Version 1.1
- IOC validation guidance
- ATT&CK mapping guidance
- Evidence appendix specification
- Confidence assessment refinements

## Version 1.2
- Threat Actor CTAR profile
- Vulnerability CTAR profile
- Campaign assessment profile
- Instructor and student example packages

## Version 2.0
- Machine-readable CTAR metadata
- STIX/TAXII mapping guidance
- CTAR textbook
- CTAR analyst workbook
- Reference evidence repository
- Community contribution program

## Long-Term Vision

Establish CTAR as an open, vendor-neutral framework for evidence-based cyber threat assessment, education, and research.
