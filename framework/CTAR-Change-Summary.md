# CTAR Framework 1.1.0 Summary of Changes

## Major Enhancements

-   Introduced modular framework architecture.
-   Added report profiles.
-   Added Historical CTAR methodology.
-   Added Analytical Framework.
-   Added ATT&CK and Defensive Mapping guides.
-   Added Historical Practice guide.
-   Added Annex Policy.
-   Standardized confidence language.
-   Standardized evidence taxonomy.
-   Clarified governance and semantic versioning.

## Backward Compatibility

Existing v1.0 CTAR reports remain valid. Historical CTARs are additive.
Annexes are optional.
